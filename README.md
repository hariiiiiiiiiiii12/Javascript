# JavaScript Notes

This repository contains my **JavaScript notes** created while learning from the **Namaste JavaScript** course by Akshay Saini.
The goal of these notes is to deeply understand how JavaScript works internally rather than just memorizing syntax.

These notes are written in **simple language**, structured for **quick revision**, and organized topic-wise.

---

## Topics Covered

### Season 1 – JavaScript Fundamentals

* Execution Context
* Call Stack
* Hoisting
* Functions & Variable Environments
* Scope Chain
* Lexical Environment
* let & const
* Temporal Dead Zone
* Closures
* Callback Functions
* Event Loop
* JavaScript Engine
* setTimeout behavior
* Higher Order Functions
* map, filter, reduce

---

### Season 2 – Asynchronous JavaScript & Advanced Concepts

* Callbacks (Good & Bad Parts)
* Callback Hell
* Inversion of Control
* Promises
* Creating Promises
* Promise Chaining
* Error Handling
* async / await
* Promise APIs

  * Promise.all()
  * Promise.allSettled()
  * Promise.race()
  * Promise.any()
* `this` keyword in JavaScript
